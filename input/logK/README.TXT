This folder contains the files with the fit coefficients for the equilibrium constants. 
It includes the following files:

logK.dat                        The standard FastChem compilation for the gas phase for elements as abundant as germanium.
                                This compilation focusses on the set of elements from the FastChem versions 1 to 3.

logK_wo_ions.dat                The standard FastChem file compilation for the gas phase but without ions.
                                This one should preferably be used at lower temperatures when ions are not
                                expected to be abundantly present.

logK_condensates.dat            The standard FastChem compilation for condensates for elements as abundant as germanium.

logK_extended.dat               The extended FastChem compilation for the gas phase for elements as abundant as lead.
                                Introduced as part of FastChem 4.

logK_extended_wo_ions.dat       The extended FastChem file compilation for the gas phase but without ions.
                                This one should preferably be used at lower temperatures when ions are not
                                expected to be abundantly present. Introduced as part of FastChem 4.

logK_condensates_extended.dat   The extended FastChem compilation for condensates for elements as abundant as lead.
                                Introduced as part of FastChem 4.


logK_hoeijmakers2019.dat        An extended list of gas-phase species. Includes ions and anions up to uranium.
                                See Hoeijmakers et al. (2019), A&A, 627, A165 for details.
